Metric Definitions

Metric Definition
Registrations The number of users who registered in this month.
Long-Term Users The number of long-term users who completed an activity during this month. A long-term user is one who has completed an activity at least 28 days after registering.
Highly-Engaged Users The number of users who completed an activity on >=4 distinct days within this month.
Note that an "activity" is currently defined as any of the following: starting a video, attempting a problem, or posting a comment, question or answer. This definition will change over time to include more diverse activities.
